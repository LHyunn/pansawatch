## 1. 개인정보의 처리 목적

본 사이트는 다음의 목적을 위하여 개인정보를 처리하며, 이용 목적이 변경되는 경우에는 사전 동의를 받습니다.

1. **회원 식별 및 인증** — Google·Kakao OAuth 를 통한 회원 가입·로그인 식별
2. **시민 투표 무결성 보장** — 1인 1표 원칙 준수, 다중 계정 어부징 탐지
3. **고지·민원 응답** — 약관 변경 통지, 오류·정정 요청 회신, 권리 행사 요청 처리
4. **서비스 운영·보안** — 비정상 접근 차단, 어뷰징·자동화 봇 식별

본 사이트는 마케팅·광고·프로파일링·제3자 광고 매칭 목적으로 개인정보를 처리하지 **않습니다.**

## 2. 처리하는 개인정보 항목

### 2-1. 이용자 (회원·투표자) 로부터 수집

| 구분 | 항목 | 수집 방식 | 비고 |
|------|------|-----------|------|
| 필수 | 이메일 주소 | OAuth 인증 (Google/Kakao) | 식별·통지 |
| 필수 | OAuth 식별자 (provider_id, sub) | OAuth 인증 | 중복 가입 방지 |
| 필수 | 표시명 (닉네임) | OAuth 기본 표시명 또는 변경값 | 운영 응대용 |
| 자동수집 | IP 주소 | 서버 액세스 로그 | 90일 후 자동 삭제 |
| 자동수집 | User-Agent (브라우저 정보) | 서버 액세스 로그 | 90일 후 자동 삭제 |
| 자동수집 | 세션 쿠키 | 인증 유지용 | 세션 종료 시 폐기 |

### 2-2. 자동 생성·수집되는 행동 데이터

| 항목 | 처리 형태 |
|------|-----------|
| 시민 투표 기록 (판결 동의/비동의, 양형 적절성, 기사 유용성) | 회원 ID 와 연결되어 저장. **탈퇴 시 회원 ID 는 익명 해시로 대체되며, 투표 자체는 익명 통계로 보존됩니다.** |

### 2-3. 처리하지 않는 항목

본 사이트는 다음의 정보를 **수집하지 않습니다.**

- 주민등록번호, 외국인등록번호 등 고유식별정보
- 휴대전화번호 (본인인증 미실시)
- 실명, 생년월일, 성별, 주소
- 신용·금융 정보
- 위치 정보 (GPS·기지국)
- 외부 광고·분석 사업자용 식별자 (Google Analytics·Meta Pixel 등 미적용)
- 프로필 이미지 (OAuth 가 제공하더라도 저장하지 않음)

### 2-4. 만 14세 미만 아동

본 사이트는 아동 대상 서비스가 아니며, 만 14세 미만 아동의 회원가입을 **허용하지 않습니다.** 가입 시 이용약관 동의 절차에서 "만 14세 이상임" 을 자기 확인하며, 14세 미만 사실이 확인된 경우 즉시 회원 탈퇴 및 관련 정보를 파기합니다.

## 3. 개인정보의 처리 및 보유 기간

| 항목 | 보유 기간 | 근거 |
|------|-----------|------|
| 이메일 / OAuth 식별자 / 닉네임 | 회원 자격 유지 기간 | 회원 식별 |
| 서버 액세스 로그 (IP, User-Agent) | 수집 후 90일 | 「통신비밀보호법」 제15조의2 및 보안 사고 대응 |
| 인증 세션 쿠키 | 세션 종료 또는 최대 30일 | 인증 유지 |
| 시민 투표 기록 | **회원 식별자는 탈퇴 시 익명화**, 투표 자체는 익명 통계로 영구 보존 | PIPA 제58조의2 (가명·익명 정보) |
| 정정 요청·민원 처리 기록 | 처리 완료일로부터 3년 | 「전자상거래법」 제6조 준용 (분쟁 대응) |

### 파기 절차

- 보유기간이 경과한 정보는 매월 1회 일괄 파기합니다.
- 전자적 파일은 복구 불가능한 방법으로 영구 삭제하며, 종이 출력물은 분쇄하여 폐기합니다.
- **탈퇴 시 익명화 처리**: 시민 투표는 통계 무결성을 위해 보존되며, 회원 식별자(user_id) 는 복구 불가능한 단방향 해시로 대체됩니다. 이는 「개인정보 보호법」 제58조의2 (가명·익명정보의 처리) 에 따른 것입니다.

## 4. 개인정보 처리의 위탁

본 사이트는 서비스 운영을 위하여 다음의 외부 사업자에게 개인정보 처리를 위탁합니다.

| 수탁자 | 위탁 업무 | 위탁 항목 | 보유 기간 |
|--------|-----------|-----------|-----------|
| Vercel Inc. (미국) | 웹 호스팅·CDN | 접속 로그, 세션 쿠키 | 위탁 계약 종료 시까지 |
| Supabase Inc. (미국, 데이터 리전 별도) | 회원 데이터베이스·인증 | 이메일, OAuth ID, 닉네임, 투표 기록 | 위탁 계약 종료 시까지 |
| Amazon Web Services (서울 리전) | 크롤러·백엔드 인프라 | (개인정보 처리 없음 — 공개 데이터 처리만) | — |
| Google LLC (미국) | OAuth 인증 (사용자 선택 시) | 이메일, OAuth sub | 인증 시점 |
| Kakao Corp. (한국) | OAuth 인증 (사용자 선택 시) | 이메일, OAuth ID, 닉네임 | 인증 시점 |

### 4-1. 위탁 변경 시 절차

새로운 수탁자가 추가되거나 기존 수탁자의 위탁 항목이 변경되는 경우, 본 사이트는 시행일 7일 이전에 본 처리방침을 개정·공지합니다.

### 4-2. 위탁 계약 시 안전성 확보

위탁 계약에는 「개인정보 보호법」 제26조에 따른 안전조치, 재위탁 제한, 손해배상 책임 조항을 포함합니다.

본 사이트의 일부 수탁자는 해외에 소재하므로 「개인정보 보호법」 제28조의8 에 따라 국외 이전 사실을 공지합니다.

| 이전받는 자 | 국가 | 이전 일시·방법 | 이전 항목 | 이용 목적 | 보유 기간 |
|-------------|------|----------------|-----------|-----------|-----------|
| Vercel Inc. | 미국 | 서비스 이용 시 실시간 (HTTPS) | 접속 로그·쿠키 | 호스팅·CDN | 위탁 종료 시까지 |
| Supabase Inc. | 미국 또는 싱가포르 (회원 가입 시 선택된 리전) | 회원 가입 시 실시간 (TLS) | 이메일·OAuth ID·닉네임·투표 | DB·인증 | 위탁 종료 시까지 |

### 5-1. 거부 권리

이용자는 국외 이전을 거부할 권리가 있으나, 거부 시 본 사이트의 회원 가입·시민 투표 기능을 이용할 수 없습니다.

## 6. 정보주체의 권리·의무 및 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

### 6-1. 권리 목록 (PIPA 제35조~제37조)

1. **개인정보 열람 요구**
2. **오류 등이 있을 경우 정정·삭제 요구**
3. **처리 정지 요구**
4. **회원 탈퇴 (동의 철회)**

### 6-2. 행사 방법

| 채널 | 절차 |
|------|------|
| 이메일 | privacy@pansawatch.org 로 신청 |
| 전용 폼 | https://pansawatch.org/privacy/request |
| 계정 설정 | 닉네임 변경·이메일 변경·탈퇴는 계정 설정 페이지에서 직접 수행 |

### 6-3. 처리 기한

- 열람·정정·삭제·처리정지 요구: 접수일로부터 **10일 이내** 조치 또는 사유 통보
- 탈퇴(계정 삭제): 접수 즉시 처리, 익명화 완료까지 최대 7일

### 6-4. 거절 사유

다음의 경우 권리 행사를 거절할 수 있으며, 사유를 명기하여 통지합니다.

- 본인 또는 정당한 대리인임이 확인되지 않는 경우
- 다른 사람의 생명·신체·재산을 부당하게 침해할 우려가 있는 경우
- 법령에서 보존 의무를 부과하는 경우 (제3-1 표 참조)

### 6-5. 본인 확인

권리 행사 시 본인 여부 확인을 위해 가입 시 사용한 이메일 주소로 인증 메일을 발송합니다.

## 8. 자동 수집 장치(쿠키)의 사용

본 사이트는 **인증 세션 유지를 위한 필수 쿠키만** 사용하며, 광고·분석·추적 목적의 쿠키는 사용하지 않습니다.

| 쿠키 | 목적 | 보존 기간 |
|------|------|-----------|
| `sb-access-token` (또는 동등) | OAuth 인증 세션 유지 | 세션 종료 시 |
| `sb-refresh-token` (또는 동등) | 세션 갱신 | 최대 30일 |
| (필요 시) CSRF 토큰 | 위변조 방지 | 세션 종료 시 |

### 8-1. 분석 도구

방문자 통계 (일 방문자 수·인기 페이지) 가 필요한 경우 **쿠키리스 익명 집계 도구** (Plausible · Umami 또는 동등 솔루션) 를 사용하며, 이는 IP 를 해시 처리하여 개인 식별이 불가능합니다. 「개인정보 보호법」 적용 대상이 아닌 익명 정보 처리에 해당합니다.

### 8-2. 쿠키 거부

이용자는 브라우저 설정을 통해 쿠키를 거부할 수 있으나, 거부 시 로그인이 필요한 기능 (시민 투표) 을 이용할 수 없습니다.

## 10. 처리방침의 변경

### 10-1. 변경 절차

본 처리방침은 법령·정책·서비스 변경 사항을 반영하여 개정됩니다.
